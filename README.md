# Themis

O sistema Themis foi desenvolvido com o propósito de oferecer um espaço seguro e confiável para a realização de denúncias relacionadas a problemas ocorridos no ambiente das instituições escolares, garantindo o anonimato dos usuários.

A denominação “Themis” tem origem na deusa da mitologia grega associada à justiça e à ordem, estabelecendo uma analogia direta com a finalidade do sistema. Assim como a deusa simboliza a justiça, a plataforma busca promover um ambiente mais justo, seguro e organizado para todos os seus usuários.

Nesse contexto, o sistema possibilita a comunicação de diferentes tipos de ocorrências, contribuindo para que todos os indivíduos se sintam ouvidos, acolhidos e protegidos no ambiente escolar.
